# PYLON ARMAMENT SELECTOR
## DESCRIPTION
This framework is a custom pylon armament selector for aircrafts with 3D camera for mission makers. It's accessible through ACE interaction menu.
- It works for both airplanes and helicopters.
- As default, it requires bomb and missile trolleys nearby the aircraft to grant access to the menu (configurable through CBA settings).
- It let's you create your own presets for each aircraft type. These can be saved in your profile.
- You can also change who owns the pylon via a button.
- Tested with vanilla and some modded aircrafts.
- MP/Dedicated compatible.

## REQUIRED ADD-ONS
- [CBA_A3](https://steamcommunity.com/sharedfiles/filedetails/?id=450814997)
- [ACE](https://steamcommunity.com/sharedfiles/filedetails/?id=463939057)

## HOW TO INSTALL
- Download the content.
- Copy and everything inside `Missionframework` and paste in your mission folder.
